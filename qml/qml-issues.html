Known Issues

1. Duo Mode Hidden Calendar Issue

In Duo mode, the Hidden Calendar may not display initially. To resolve this:

  1. Create a link in No Entry mode first.
  2. Reload the browser.
  3. Insert the existing calendars into both fields (even if Hidden Meetings is not among them).
  4. Save the Duo settings.

After saving, the Hidden Calendar should now be visible.

IF IT DIDN'T HELP:

  1. Try logging out of the extension by clicking logout.
  2. Then, on the page chrome://extensions, click the "Refresh" button.
  3. Check the Duo settings again.
  4. If nothing works, feel free to contact me on Telegram: @acoola72.

2. Duo Template Link Duplication

In the Duo template settings, parameters may duplicate, and the preview will not be visible until the user types a character in another block. This is not critical, as the input in the Duo link template will still be saved when the button is pressed.

3. Access Issues After Logout

If you encounter issues after logging out of the extension, such as lost access or inability to retrieve calendars, try the following:

  1. Open the chrome://extensions,
  2. Go to the Quick Meeting Links > Debug page.
  3. Find caption "Service Workers" and click on them.
  4. Execute logout() in the console to force a logout.

This should help you regain access to the extension.
